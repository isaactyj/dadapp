How Support Is Estimated

Recent pivot lows, rolling lows, and moving averages are clustered, then the nearest floor under price is chosen.

How Take Profit Is Estimated

Nearby resistance clusters are used first. If price is already above them, the app falls back to a measured-move style target.

What Qualifies

A hit means the post-earnings close beat the return threshold you set when compared with the price at the start of the pre-earnings window.

Important Limitation

This is an approximation using Yahoo historical earnings records. It is useful for screening, but not a perfect event-study engine.
